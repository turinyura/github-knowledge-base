# 📚 GitHub Knowledge Base

Интерактивный справочник по работе с Git и GitHub для эффективного взаимодействия с ИИ-агентами.

## 🎯 Назначение

Этот репозиторий служит учебным полигоном и базой знаний. Здесь собраны структурированные материалы по настройке Git, рабочим процессам (workflow) и правилам взаимодействия с ИИ-агентами.

### 🟢 Основы (Basics)
- [Терминология](./01_Basics/terminology.md) — ключевые понятия Git и GitHub.
- [Настройка окружения](./01_Basics/README.md) — установка и конфигурация инструментов.

### 🟡 Рабочий процесс (Workflow)
- [Ветки и слияния](./02_Workflow/README.md) — управление ветками, merge и rebase.
- [Pull Requests](./02_Workflow/README.md) — создание и ревью PR.

### 🤖 ИИ-Агент (Agent Rules)
- [Правила промптов](./03_Agent_Rules/README.md) — гайдлайны для эффективной работы с ИИ.

## 🚀 Как пользоваться

1. **Перемещение**: Используйте ссылки выше для навигации по разделам.
2. **Поиск**: Нажмите `Ctrl+F` (или `Cmd+F` на Mac) для быстрого поиска по тексту в рамках текущего файла.
3. **Структура**: Материалы организованы по папкам согласно тематическим разделам.

## 🤝 Участие

Правки в базу знаний вносятся через **Pull Requests**. Если вы ИИ-агент, следуйте правилам из раздела [Agent Rules](./03_Agent_Rules/README.md) при создании изменений.
