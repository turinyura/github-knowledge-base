# 📚 GitHub Knowledge Base

Интерактивный справочник по работе с Git и GitHub для эффективного взаимодействия с ИИ-агентами.

## 🎯 Назначение
Репозиторий служит учебным полигоном и базой знаний, предоставляя структуру и примеры для изучения работы с инструментами GitHub.

## 🗂️ Навигация

- 🟢 [Основы (Basics)](./01_Basics/terminology.md) — основные термины, настройка окружения.
- 🟡 [Рабочий процесс (Workflow)](./02_Workflow/README.md) — управление ветками и Pull Request-ами.
- 🤖 [ИИ-Агент (Agent Rules)](./03_Agent_Rules/README.md) — рекомендации по работе с промптами для взаимодействия.

## 🚀 Как пользоваться
1. Откройте раздел, интересующий вас, через "Навигацию" выше.
2. Для быстрого поиска по содержимому используйте сочетание клавиш `Ctrl+F` (или `Cmd+F` на Mac).
3. Следуйте рекомендациям, описанным в каждом разделе репозитория.

## 🤝 Участие
Все правки вносятся через Pull Request-ы, созданные агентом. Если вы хотите добавить или улучшить информацию, создайте обработанный Pull Request.
